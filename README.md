# 曾国藩个人网站

## 项目简介

这是一个关于晚清中兴名臣曾国藩的综合性个人网站，致力于传播曾国藩的思想文化，弘扬中华优秀传统文化。网站内容丰富，包含曾国藩的生平事迹、思想理念、著作文献、名言精选等多个方面。

## 主要功能

### 🏠 首页
- **英雄区域**：展示曾国藩的核心形象和简介
- **核心价值**：突出曾国藩的修身智慧、治家理念、识人用人、领导力等核心价值
- **快速导航**：提供便捷的页面导航功能

### 📚 生平事迹
- **时间线展示**：以时间线的形式展示曾国藩的重要人生节点
- **详细描述**：包含出生、中进士、创建湘军、攻克天京、逝世等重要事件
- **历史背景**：结合晚清历史背景，展现曾国藩的历史地位

### 🧠 思想理念
- **理学思想**：继承程朱理学，强调修身养性
- **经世致用**：主张学问要用于实践，反对空谈
- **齐家思想**：重视家庭教育，强调勤俭持家
- **修身理念**：提出"慎独"、"主敬"、"求仁"、"习劳"等修身方法

### 💬 名言精选
- **经典语录**：收录曾国藩的经典名言和格言
- **分类展示**：按不同主题分类展示名言
- **详细解释**：提供名言的背景和含义解释

### 📖 著作文献
- **《曾国藩家书》**：修身、齐家、教育、为官等方面的书信集
- **《冰鉴》**：相术著作，阐述识人用人的智慧
- **《挺经》**：修身著作，阐述修身理念和方法
- **《曾国藩日记》**：记录日常生活和思想变化
- **《曾文正公全集》**：收录各类作品的总集
- **《经史百家杂钞》**：古文选集，体现文学思想

### 🏛️ 历史影响
- **政治影响**：平定太平天国，挽救清王朝统治
- **文化影响**：理学思想和修身理念对后世的影响
- **教育影响**：培养大批人才，如李鸿章、左宗棠等
- **思想影响**："经世致用"思想对近代思想史的影响

### 🏮 本地化内容
- **出生地与成长环境**：湖南湘乡荷叶塘的地理和人文环境
- **湘军与湖南精神**：体现湖南人"吃得苦、霸得蛮、耐得烦"的精神特质
- **湖南文化的影响**：湖南深厚的文化底蕴对曾国藩思想形成的影响
